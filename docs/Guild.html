createChannel(name, typeopt, permission_overwritesopt, topicopt, bitrateopt, user_limitopt, parent_idopt, nsfwopt, toObjectopt) → {Promise.<(Channel|object)>}

Creates a channel on the guild.

Parameters:
Name Type Attributes Default Description
name string

Name of the channel to be created.
type string <optional>
 text

Type of the channel to be created, either 'text', 'voice', 'category'.
permission_overwrites Array <optional>
 []

The channel's permission overwrites.
topic String <optional>

The channel's topic (0-1024 characters).
bitrate Number <optional>

The bitrate (in bits) of the voice channel (voice only).
user_limit Number <optional>

The user limit of the voice channel (voice only).
parent_id String <optional>

The id of the parent category for a channel.
nsfw Boolean <optional>
 false

Whether the channel is nsfw.
toObject boolean <optional>

Send the data as a JSON Object instead of a class.
